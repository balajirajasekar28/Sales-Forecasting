Sales Forecasting Project:
A simple end-to-end Sales Forecasting project using Python, MySQL, Pandas, and Machine Learning.
This project analyzes sales data, performs data cleaning, generates visual insights, and builds a basic prediction model for future sales trends.

Project Overview:
This project connects to a MySQL database, loads sales data, cleans it, visualizes key metrics, and predicts sales using a linear regression model.
It also exports cleaned data and supports business decision-making through analytics.

Technologies Used:
Python
MySQL + SQLAlchemy
Pandas & NumPy
Matplotlib & Seaborn
Scikit-Learn
Power BI (for dashboard visualization)

Features:
Import sales data directly from MySQL
Clean and preprocess raw data
Exploratory Data Analysis (EDA)
Category-wise and city-wise sales insights
Monthly sales trend visualization
Simple Linear Regression model for forecasting
Export cleaned dataset as CSV
Power BI dashboard for interactive insights
